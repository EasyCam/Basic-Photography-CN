# 项目概述

这个项目最开始的定位是一个读书笔记而已。

原来是五年前对[Langford&#39;s Basic Photography](https://www.sciencedirect.com/science/book/9780240521688)的读书笔记。
后来发现教科书内容总是不那么灵活，因此也就不局限于教材本身，准备从更多方面来收集资料。

现在就尝试着将其改成一个类似简单指南的探索项目吧。

# 1. 拍照之前

拍照之前，我们需要聊一聊一些背景话题。

关于这些事情，以及本项目相关的所有事情，大家都完全可以有自己的观点和看法。

君子和而不同，这很正常。

## 1.1 时代的趋势

视觉是人类获取信息的主要途径，而出于社会生活习性，人又有分享的冲动。

把所见的场景固定下来，分享给更多人看，曾经是很困难的。

古代乃至一直到近代，都需要有专门培训过的技术人员进行绘制。

摄影术的诞生，让人类终于可以将一段时间内的景象固定下来，给更多人观看了。

后来人们发现，摄影不仅能记录场景，还能对时间进行冻结或者延长。

用高速快门可以查看奔跑中的马蹄到底是什么样子。

用长时间曝光可以看到天空中肉眼难以发现的星体。

上世纪末，摄影在本土都是相对稀罕的事物，和大众的日常生活总有那么点距离。

上世纪七十年代以后，本土有了“四月影会”，以及后来的“西北风摄影”。

其中的很多作品，都是将普通人作为拍摄对象，于是有人说这些都是“平民视角的摄影”。

果真如此？

恐怕未必。

无论是“四月影会”中的文艺青年，还是“西北风摄影”当中的先驱人物，恐怕更多还是将镜头对准平民。

平民此时此刻是作为“客体”被人“欣赏”，被人从自己的生活中攫取一滴滴闪光的瞬间，然后连缀成符合“文化人”口味的精美饰品。

好在时代在发展，技术在进步，数字时代，拍照的门槛大大降低了。

1930年代，曹元宇老先生的《摄影术》当中有大量的化学相关内容作为背景知识；

1970年代，Don Sheff 买下纽约摄影学院，创作系列教材，其中的化学知识已经很少了；

2010年代，市面上绝大多数摄影教材都基本以数码摄影作为主要内容了。

胶片摄影，不仅有胶片本身的物料成本，还有技术层面复杂度带来的学习成本、拍摄成功率存在不确定性的机会成本、拍摄后冲洗放大等过程的额外流程成本等等。

数码摄影，日趋廉价的存储卡替换掉了昂贵的胶片，数字图像处理技术也提供了更多的容错空间，借助互联网尤其是移动互联网而实现的图片分享更是降低了传播成本。

属于民众的摄影，至少要从数码时代开始。

真正的平民视角，是广大民众作为主体，从自己的目光出发，去审视和欣赏整个世界。

然而，民众选择的是短视频。

这也正常，毕竟声画结合能带来更丰富的感官体验，配上一些荒诞的故事或者耸人听闻的剧情更是传播利器。

而静态图片的摄影，现在看来，有点淡出流行的感觉了。

精心拍摄的照片，做好前期准备，再做好后期处理，上传到社交网络，未必有多少关注和点赞。

如果是同样时间内拍一个极其低成本的放歌曲对口型的视频，可能就会收获到几倍甚至更多的流量。

一方面是大众传播平台往往流行短平快的浅层次阅读体验，另一方面是这些平台也有对应的推荐机制和偏好。

要做好这样的心理准备，不要有太高的心理预期。

## 1.2 拍照的动机

好吧，无论如何，我们都选择了拍照了。

拍照之前，先问一下自己，为什么要拍照，要给什么对象或者场景拍照，想拍什么样的照片？

当然，这并不需要立刻回答得特别清楚。

首先是因为人做事情未必总想得特别清楚，有时候是随心所欲，有时候是随遇而安，也有时候是随波逐流。

另外就是人心善变，最初的梦想在后面可能也会被放下或者修改，最初的立场后来也可能会发生变更等等。

但总还是得有个大概的初步认知才行。

因为这一部分的答案，决定了你在当前阶段该如何选择装备。

人各有志，选择同样的事物往往也可能有截然不同的动机。

动机可能不太好明确，但落实到若干场景分类，就方便多了。

这些不同场景都有各自适合的器材组合。

下面这部分内容也就是个大概，其中一些术语内容如果读者暂时不太明白也不要紧，以后慢慢就都知道了。

### 1.2.1 拍家人朋友

在上世纪末或者本世纪初期，民众买相机的最常见动机是给家人拍照留念。

那时候去照相馆拍照价格很昂贵，旅游景区拍照都可能要十块二十块的价格。

拍照给家人留念的，往往是追求性价比，未必对摄影本身有多大的热爱。

二十多年前，我老爸从县城里花了180块买了一台的腾马便携相机，给家人拍照。

那台相机的型号好像是M800，使用135胶卷，自带了35mm镜头，F/3.5固定光圈。

除了相机以外，胶片本身和冲洗费用也都是持续不断的成本。

后来我老爸觉得40块钱一卷的富士胶卷实在太贵，就没再坚持下去。

那时候的单反相机可能要更贵，节俭年代的人也很难舍得用那么高成本去拍家人朋友。

但实际上用单反能有更好的拍摄体验，哪怕是真正自动对焦的便携相机也比这种泛焦镜头的体验更好。

家庭环境往往是在室内，照明未必很理想，固定光圈的泛焦镜头画质就总是不太理想。

如果我能穿越回去，一定会建议他买二手的美能达或者宾得的自动单反套机。

那时候都180块人民币相当于现在多少钱呢？就拿当时我爱吃的土豆丝卷饼来比一下吧。

二十多年前，在我读高中的时候，买一个不加鸡蛋的土豆丝卷饼要五毛钱，加一个鸡蛋就要一块钱。

今天下午（2023年12月11日），我碰巧又买了一个差不多同规格的土豆丝卷饼，也加了一个鸡蛋，花了十块钱。

那从土豆丝卷饼加鸡蛋来粗略类比，当时的180块，大概相当于现在的1800块。

现在用1800块，大概只能买一些拍立得或者超小画幅的便携相机，前者后续成本高，后者画质极差，都不太划算。

当然也能买到一套二手的索尼 NEX 微单相机，或者其他牌子的二手入门级的单反相机。

室内拍摄人像，往往要考虑镜头焦距和光圈，前者限制视野和拍摄距离，后者影响可用快门速度。

特殊情况是给小朋友拍照，怕晃到孩子就不太好用闪光灯，得尽量用高感光度、高速快门。

其他人物在一般室内场景拍照，考虑到手持稳定性和画面范围，也不适合太长焦距镜头。

综合来看，使用24-105mm F/4或者24-70mm F/2.8等类似参数的变焦镜头比较方便。

### 1.2.2 拍学校生活

在本世纪初期，我这一代人就开始读大学了，来到大学发现有的专业需要用相机。

比如各种专业的大学生参加实习或者调查研究，总要用些照片。

这种场景，往往是一个个小分队从所在院校或者系部借出各种相机，然后就对付着用。

借来的相机终究不如自己买来的用着舒坦，有一些同学在勤工俭学或者得奖学金后就会自己买一台。

当然了，这个时期的数码单反相机价格高昂，胶片相机门槛又高，很多人选择的都是一些小尺寸卡片机。

十多年前，在我读大学的时候，跑到中关村给人装电脑，一台机器能赚五十块，就这么慢慢熬。

攒了一段时间，500块买了一台爱国者的 V1000 卡片相机，1/1.8英寸1037万像素 CCD，三倍光学变焦。

我拿着这台相机拍了一段时间的野外地质现象，还尝试去拍摄矿物样本，以及显微镜下的矿物薄片等等。

也正是这段时间的拍摄经历，让我体验了卤钨灯、LED灯等不同光源的光谱连续性差异。

后来在拍金属硫化物的过程中，又探索了红外摄影，还试了外置偏光拍摄方案。

当然了，有了第一台数码相机，自然地也会拿来拍摄身边的同学。

当时同学们对于我拍摄他们还都有些抵触，一方面是有点害羞，另一方面是觉得我拍摄的都是生活场景不够好看。

但是十多年后，他们再次看到这些照片，就往往感慨韶华易逝岁月无情，又遗憾当年没有拍更多的照片。

学校里面普遍比较明亮，宿舍教室图书馆等等大概都有严格的照明标准，所以未必需要太高感光度和大光圈。

一般情况下，各种相机的套头（APS-C画幅的18-55、135画幅的28-70等等）基本足够。

如果希望能够去除掉复杂背景信息带来的干扰，也可以额外添加一些能突出主体又缩减背景视角的长焦镜头。

### 1.2.3 拍风光景物

人怕入错行，之前的行当需要四海为家，四处漂泊。

也许是由于所学内容关注的都是几个百万年以上的时间尺度，当时对眼前的山川风光没有太多动心之处。

同行的一些老师同学倒是有人很喜欢拍摄这些内容，天山长河，大漠瀚海，乡村田亩，都市街头。

很多景物在当时直观看起来很震撼，拍出来的东西却总觉得差点意思，现在回想起来，是当时的设备宽容度太低了。

也有的老师同学能拍到很精致的画面，那些画面往往需要他们很早动身，赶在日出日落之前，只有很窄的窗口期。

有的地方山水风光清秀出众，有的地方民居建筑历史悠久，也有的地方钢铁森林壮观震撼。

对于有的学科来说，摄影也是一种重要的工具，风光和景物本身可能只是信息的载体。

杜塞尔多夫艺术学院的**贝歇夫妇** （Bed & Hilla Becher） 曾经专门拍摄人类的工业建筑，北京工业大学的杨晓利老师四处拍存留下来的古塔。

风光景物可能需要广角镜头，也可能需要长焦镜头。

广角镜头是为了近距离拍摄大范围，尽量多容纳背景信息；长焦镜头是为了进一步突出主体，尽量减少背景信息。

这两种情况，都可能发生。所以使用28-200、28-300、24-240之类的大变焦镜头就是一个不错的选择。

这类镜头也被称为“一镜走天下”的“天涯镜”，而且画质通常未必多出色。

但摄影最重要的是拍到内容，画质其实是相对比较次要的事情。

很多拥有高画质设备的器材爱好者，未必拍了什么有意思的内容。

而如果反过来你拍的内容足够有趣，画质差点也还是能忍的。

内容层面上是有无的问题，画质层面上只是精细程度的问题。

当然了，如果你要拍一些特殊的场景，还是得有对应的特殊设备：

* 拍摄星野（天上星空和地面景观兼顾），可能就得有大光圈超广角和稳定的三脚架；
* 拍摄深空（星系、星云之类的），可能就得有大口径长焦镜头，还得有赤道仪之类设备；
* 拍摄星轨（就是地面景观加上天上星星的轨迹），可能得有支持长曝光的快门遥控器。

### 1.2.4 拍体育动物

从拍摄过程来看，拍摄运动员和拍摄动物的难度差不多。

首先，你必须要对拍摄对象有足够的了解:

* 要知道场上运动员的角色分工；
* 要知道所拍摄动物的生活习性。

然后，拍摄的时候要注意多用高速度连拍：

* 运动厂商瞬息万变，一旦错过没法补拍；
* 动物不会配合拍摄，也要尽量捕捉画面。

另外，拍摄时候还往往要保持充足物理距离：

* 运动员的体育赛事怕被干扰，不能离太近；
* 动物可能害怕或者产生敌意，也不能太近。

所以，拍摄体育赛事和野生动物，往往要用长焦镜头，对焦速度要快，机身连拍也要快。

早些年的各种所谓“新闻机”，就是专门给新闻记者设计的，以对焦和连拍速度见长。

镜头方面，你可能需要 70-200mm F/2.8 这种兼具中长焦的大光圈镜头，或者是 150-600mm F/5-6.3 之类的大长焦镜头。当然上面这种镜头都很重，

有的镜头可能焦段更长，而且画质更好，比如 300mm F2.8、600mm F/4等长焦定焦镜头，但这种镜头都特别重。实际上用起来不太方便，往往需要在场地支起三脚架，至少也得有独脚架才方便移动。

要兼顾轻便和长焦，可能就得用 100-300mm F/4.5-5.6 APO 这种镜头，当然画质和对焦速度就可能逊色一点。
